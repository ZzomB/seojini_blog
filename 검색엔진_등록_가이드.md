# 검색엔진 등록 가이드

## 📋 개요
구글과 네이버에서 블로그가 검색되도록 하기 위한 단계별 가이드입니다.

---

## 🔍 구글 검색엔진 등록

### 1단계: Google Search Console 가입
1. [Google Search Console](https://search.google.com/search-console) 접속
2. Google 계정으로 로그인
3. "속성 추가" 클릭

### 2단계: 속성 추가 방법 선택
**방법 1: 도메인으로 추가 (권장)**
- 도메인 전체를 등록 (예: `yourdomain.com`)
- 모든 하위 도메인 포함

**방법 2: URL 접두어로 추가**
- 특정 URL만 등록 (예: `https://yourdomain.com`)
- 더 간단한 인증

### 3단계: 사이트 소유권 확인
**HTML 파일 업로드 방법:**
1. Search Console에서 제공하는 HTML 파일 다운로드
2. 프로젝트의 `public` 폴더에 업로드
3. 사이트에 접근 가능하도록 배포
4. Search Console에서 "확인" 클릭

**HTML 태그 방법:**
1. Search Console에서 제공하는 메타 태그 복사
2. `app/layout.tsx`의 `<head>` 섹션에 추가
   ```tsx
   <head>
     <meta name="google-site-verification" content="YOUR_VERIFICATION_CODE" />
   </head>
   ```
3. 또는 `metadata` 객체에 추가:
   ```tsx
   other: {
     'google-site-verification': 'YOUR_VERIFICATION_CODE',
   }
   ```

### 4단계: Sitemap 제출
1. Search Console 대시보드에서 "Sitemaps" 메뉴 선택
2. Sitemap URL 입력: `https://yourdomain.com/sitemap.xml`
3. "제출" 클릭
4. 몇 시간 내에 크롤링 시작

### 5단계: robots.txt 확인
- 현재 설정: `/robots.txt`에서 확인 가능
- 모든 검색엔진 허용: `User-agent: *` / `Allow: /`
- API 경로 차단: `Disallow: /api/`

## 🔍 네이버 검색엔진 등록

### 1단계: 네이버 서치어드바이저 가입
1. [네이버 서치어드바이저](https://searchadvisor.naver.com/) 접속
2. 네이버 계정으로 로그인
3. "웹마스터 도구" 메뉴 선택

### 2단계: 사이트 등록
1. "사이트 추가" 클릭
2. 사이트 URL 입력 (예: `https://yourdomain.com`)
3. 사이트 정보 입력 (사이트명, 카테고리 등)

### 3단계: 사이트 소유권 확인
**HTML 파일 업로드 방법:**
1. 서치어드바이저에서 제공하는 HTML 파일 다운로드
2. 프로젝트의 `public` 폴더에 업로드
3. 사이트에 접근 가능하도록 배포
4. 서치어드바이저에서 "확인" 클릭

**메타 태그 방법 (권장):**
1. 서치어드바이저에서 제공하는 메타 태그 복사
   - 예: `<meta name="naver-site-verification" content="YOUR_CODE" />`
2. 환경 변수 파일 (`.env.local`)에 추가:
   ```env
   NAVER_SITE_VERIFICATION=YOUR_VERIFICATION_CODE
   ```
3. 이미 `app/layout.tsx`에 설정되어 있음:
   ```tsx
   other: {
     'naver-site-verification': process.env.NAVER_SITE_VERIFICATION || '',
   }
   ```
4. 서치어드바이저에서 "확인" 클릭

### 4단계: Sitemap 제출
1. 서치어드바이저 대시보드에서 "요청" > "사이트맵 제출" 선택
2. Sitemap URL 입력: `https://yourdomain.com/sitemap.xml`
3. "확인" 클릭
4. 제출 후 상태 확인

### 5단계: RSS 제출 (선택사항)
- 블로그에 RSS 피드가 있다면 제출 가능
- 네이버는 RSS를 통한 빠른 수집 지원

## 🔧 환경 변수 설정

`.env.local` 파일에 다음 변수들을 설정하세요:

```env
# 사이트 URL (필수)
NEXT_PUBLIC_SITE_URL=https://yourdomain.com

# 네이버 사이트 인증 코드 (선택)
NAVER_SITE_VERIFICATION=your_naver_verification_code

# 구글 사이트 인증 코드 (선택, 필요시)
GOOGLE_SITE_VERIFICATION=your_google_verification_code
```

## 📊 검색엔진 등록 후 확인 사항

### 구글
1. **인덱싱 상태 확인**
   - Search Console > "URL 검사"에서 특정 페이지 검색 가능 여부 확인
   - "색인 생성" 요청 가능

2. **성능 모니터링**
   - "성능" 탭에서 검색 노출 수, 클릭 수 확인
   - 평균 노출 위치 확인

3. **일반적인 대기 시간**
   - 사이트맵 제출 후: 1-3일
   - 첫 인덱싱: 3-7일
   - 검색 결과 노출: 1-2주

### 네이버
1. **수집 현황 확인**
   - 서치어드바이저 > "수집 현황"에서 확인
   - 수집된 페이지 수 확인

2. **검색 노출 확인**
   - 네이버에서 `site:yourdomain.com` 검색
   - 인덱싱된 페이지 확인

3. **일반적인 대기 시간**
   - 사이트맵 제출 후: 1-2일
   - 첫 수집: 3-5일
   - 검색 결과 노출: 1주-2주

## 🚨 문제 해결

### 구글에서 검색이 안 될 때
1. **robots.txt 확인**: 크롤러 차단 여부 확인
2. **사이트맵 상태**: Search Console에서 오류 확인
3. **페이지 인덱싱 요청**: "URL 검사"에서 직접 요청
4. **콘텐츠 품질**: 중복 콘텐츠, 저품질 콘텐츠 확인

### 네이버에서 검색이 안 될 때
1. **사이트 인증 상태**: 서치어드바이저에서 확인 완료 여부 확인
2. **사이트맵 오류**: 제출한 사이트맵에 오류 없는지 확인
3. **수집 현황**: "수집 현황"에서 수집된 페이지 수 확인
4. **콘텐츠 정책**: 네이버 정책 위반 여부 확인

## 📝 추가 최적화 팁

### 1. 구조화된 데이터 추가
게시글에 JSON-LD 형식의 구조화된 데이터를 추가하면 검색 결과에 리치 스니펫이 표시될 수 있습니다.

### 2. 이미지 최적화
- 이미지에 적절한 alt 텍스트 추가
- 이미지 파일명을 의미있게 설정
- 이미지 크기 최적화

### 3. 내부 링크 최적화
- 관련 게시글 간 내부 링크 추가
- 의미있는 앵커 텍스트 사용

### 4. 정기적인 콘텐츠 업데이트
- 정기적으로 새 게시글 작성
- 기존 게시글 업데이트
- 사이트맵 자동 업데이트 확인
